Command-and-control (C&C) server

A server that issues commands to compromised devices and receives data from them.

Description

In the context of cybersecurity, a Command-and-control (C&C) server is a central server that is used by cybercriminals to maintain communications with compromised systems within a network. These servers allow attackers to execute remote commands on infected devices, which can include deploying malware, stealing sensitive information, or orchestrating further attacks. The C&C server acts as a control center for botnets—networks of infected computers that can be commanded as a single entity. C&C servers can communicate with infected devices using various protocols, including HTTP, HTTPS, and IRC, often disguised within legitimate traffic to evade detection. The use of C&C servers is a critical aspect of many cyber threats, including ransomware, spyware, and distributed denial-of-service (DDoS) attacks. Cybersecurity professionals work to identify and disrupt these servers as part of their defense strategies to protect networks and systems from ongoing threats.

Examples

  • A botnet controlled by a C&C server to launch DDoS attacks on targeted websites.
  • Ransomware that communicates with a C&C server to receive encryption keys.

Additional Information

  • C&C servers can be hosted on various platforms, including cloud services, making them harder to trace.
  • Law enforcement agencies frequently target these servers to dismantle cybercrime operations.
