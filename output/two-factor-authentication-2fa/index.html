Two-factor authentication (2FA)

A security process that requires two different forms of identification to access an account.

Description

Two-factor authentication (2FA) is a security measure used to enhance the protection of accounts and sensitive data. It requires users to provide two different authentication factors to verify their identity before granting access. This method significantly reduces the risk of unauthorized access, as it combines something the user knows (like a password) with something the user has (such as a smartphone or hardware token) or is (biometric verification). The first factor is typically a password, while the second factor can be a code sent via SMS, an authentication app, or a fingerprint scan. 2FA is widely adopted across various online services, including email providers, social media platforms, and banking institutions. The goal of 2FA is to ensure that even if a password is compromised, an attacker would still need the second factor to gain access, thereby providing an extra layer of security against phishing attacks and other common threats in the cybersecurity landscape.

Examples

  • Receiving a text message with a verification code after entering your password.
  • Using an authentication app like Google Authenticator to generate a time-sensitive code.

Additional Information

  • Many services offer 2FA options, including Google, Facebook, and financial institutions.
  • 2FA can be implemented using various methods, including SMS, email, authenticator apps, or biometric factors like fingerprints or facial recognition.
